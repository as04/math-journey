The mental model to lock in today

A matrix is a function that transforms space. It takes vectors to vectors. Linearly. Nothing more, nothing less.

  • The columns of M tell you where the basis vectors î = (1,0) and ĵ = (0,1) land after the transformation. Everything else is determined by linearity.
  • det(M) = signed area-scaling factor. Negative = orientation flipped. Zero = collapses space onto a line.
  • Eigenvectors = directions that the transformation only stretches, doesn't rotate. Eigenvalues = the stretch factors.
  • Matrix multiplication = composition of transformations.

Things to try in the visualizer

  1. Diagonal matrix [[2,0],[0,3]]: stretches î by 2, ĵ by 3. Determinant 6. Both basis vectors are eigenvectors.
  2. Shear [[1,1],[0,1]]: pushes the top of the unit square sideways. Determinant still 1 (area preserved). Eigenvalues both 1 — only the x-axis is an eigen-direction.
  3. Rotation: complex eigenvalues — there's NO real direction that a rotation leaves alone.
  4. [[1,2],[2,4]]: determinant 0 — space collapses onto a line. This is a singular (non-invertible) matrix.
  5. Negative determinant: try [[1,0],[0,-1]] — reflects across the x-axis. The shaded unit square flips orientation.

Why this matters for ML

Every neural network layer is (mostly) a matrix multiplication. Every feature transformation in classical ML is a matrix. PCA finds eigenvectors of the covariance matrix. SVD is the most-used decomposition in numerical ML. Singular (det=0) matrices break things — that's why "condition number" matters in practice. None of this lands properly unless the geometric picture is in your head first.
